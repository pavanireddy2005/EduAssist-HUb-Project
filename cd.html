Cloud computing delivers computing resources (servers, storage, databases) over the internet.

  • Pay-as-you-go pricing
  • On-demand scalability
  • No physical hardware needed

Cloud services are categorized based on the level of control and management.

  • IaaS – Infrastructure as a Service
  • PaaS – Platform as a Service
  • SaaS – Software as a Service

Defines how cloud infrastructure is managed and accessed.

  • Public Cloud
  • Private Cloud
  • Hybrid Cloud
  • Community Cloud

AWS is the most widely used cloud provider with global infrastructure.

  • EC2 – Virtual Machines
  • S3 – Object Storage
  • RDS – Managed Databases
  • VPC – Virtual Private Cloud

DevOps combines development and operations to improve software delivery speed and quality.

  • Continuous Integration
  • Continuous Testing
  • Continuous Delivery

CI/CD automates the build, test, and deployment cycle.

  • Code → Build → Test → Deploy
  • GitHub Actions / Jenkins
  • Automated environment provisioning

Docker packages applications into portable containers.

  • Lightweight and fast
  • Environment consistency
  • Easy deployment

Kubernetes is a container orchestration system for managing clusters.

  • Auto-scaling
  • Load balancing
  • Self-healing containers
