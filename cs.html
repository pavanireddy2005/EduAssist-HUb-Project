Cyber Security protects systems, networks, and data from digital attacks.

  • Protects sensitive information
  • Prevents unauthorized access
  • Ensures data integrity and confidentiality

Threats are potential dangers that can exploit vulnerabilities.

  • Malware
  • Phishing
  • Ransomware
  • Social engineering

Cyber attacks are deliberate attempts to steal, damage, or disrupt systems.

  • DDoS Attacks
  • SQL Injection
  • Man-in-the-Middle (MITM)
  • Brute Force
  • Zero-day exploits

Network Security ensures safe data transfer across networks.

  • Firewalls
  • Intrusion Detection Systems (IDS)
  • Intrusion Prevention Systems (IPS)
  • VPN Encryption

Cryptography protects information by converting it into unreadable format.

  • Encryption & Decryption
  • Symmetric Algorithms
  • Asymmetric Algorithms
  • Hashing

Tools used to monitor and secure systems.

  • Nmap
  • Wireshark
  • Metasploit
  • Burp Suite
  • OWASP ZAP

Security policies guide safe use of systems and data.

  • Password policy
  • Data privacy policy
  • Incident response plan
  • Access control rules

Follow recommended practices to stay secure online.

  • Use strong passwords
  • Enable multi-factor authentication
  • Keep software updated
  • Avoid unknown links & downloads
